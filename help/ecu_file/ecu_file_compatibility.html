Kompatibilitätsliste

Um die Daten eines µC abzugreifen ist es notwendig das Labskaus die Informationen über die ECU korrekt erfassen kann. Mit der Zeit wurde Labskaus immer weiter entwickelt und ist hierbei auf hoffentlich immer besser geworden. Diese Evolution hat leider zur Folge dass ECU-Beschreibungsdateien, welche von mit einer früheren Version erstellt wurden, nicht mehr mit der neusten Version von Labskaus kompatibel sind. Die folgende Tabelle gibt eine Übersicht welche Versionen miteinander kompatibel sind.
