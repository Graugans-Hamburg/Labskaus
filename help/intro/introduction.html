Einleitung

Problem

Labskaus dient dazu Variablen eines Mikrocontrollers auszulesen. Dieses sollte auf eine möglichst generische Art erfolgen. Während der Entwicklung eines µC Programms kommt es immer wieder dazu das man sich den Wert einer Variable live anschauen möchte. Extrem unpraktisch wäre es das Programm so zu verändern das die entsprechende Variable auf einer Schnittstelle ausgegeben wird, die Ausgabe dann auf dem PC anzufangen und einzeln auszuwerten. Bei einer Variable geht das vielleicht noch aber dann stellt man fest das man noch mehr Variablen braucht und alles fängt an Unschön zu werden.

Lösung

Deshalb wurde das Programm Labskaus geschrieben. Es übernimmt all die oben genannten Aufgaben und das auf eine so generische Art und Weise das das µC-Programm nicht ständig umgeschrieben werden muss. Die Kommunikation erfolgt hierbei über die UART Schnittstelle. Das Programm Labskaus sendet Anfragen an den µC und erfragt hierbei die Werte der einzelner Speicherzellen und wertet die Antworten des µC aus.
