Kommunikationsschnittstelle einstellen

Die Schnittstelle die Labskaus verwendet kann über das Dialogfenster eingestellt werden welches über die Menüleiste (Settings) geöffnet werden kann. Eingestellt werden kann:

  • Port Number (z.B. COM1, /dev/ttyUSB0). In dem entsprechenden Eingabefeld kann kein Nummer einge gegeben werden. Jede Nummer entspricht immer einer Schnittstelle welche noch von dem verwendeten Betriebssystem abhängt. Als Beispiel: Wenn die Nummer 1 eingegeben wird so würde, wenn das Programm auf einem Windows PC ausgeführt wird, die Schnittstelle COM 1 verwendet. Auf einem Linux System entspricht diese Nummer dem Port /dev/ttyS0. Ein Auflistung aller Portnummern und deren Ports kann in der folgenden Tabelle eingesehen werden.


  • Baudrate - ich denke hier ist kein weiteres Wort notwendig


  • Mode bestimmt die weiteren Eigenschaften der seriellen Schnittstelle wie z.B. die Verwendung von Stopbits oder eine Paritybits. Die möglichen eingaben sind in der Tabelle weiter unten aufgelistet.
