Installation Linux (ubuntu 14.04)

Kurzanleitung

  • Herrunterladen der Software von http://graugans.ddns.net
  • Entpacken der Software in einen beliebigen Ordern (Empfehlung /opt/)
  • Das Skript: configure.sh ausführen
  • Fertig

Erläuterung

Das Installationspaket enthält die ausführbare, bereits kompilierte Datei Labskaus welche nach der Durchführung der Installationsanleitung direkt ausgeführt werden kann. Neben der ausführbaren Datei enthält das Installationspaket ebenfalls die zugehörige Programmhilfe in Form von html-Dateien.

Das Skript configure.sh ist notwendig um die Abhängigkeit aufzulösen. Die grafische Oberfläche des Programm Labsakus wurde mit den Bibliotheken des wxWidgets-Frameworks erstellt. Diese Bibliotheken werden mit dem Skript installiert. Aufgrund dieser Installation muss es mit Adminrechten ausgeführtwerden.

Installation Windows

Einfach den das Zip file entpacken, an einen Beliebigen Pfad kopieren. Zum Starten des Programms einfach auf Labskaus.exe klicken.
