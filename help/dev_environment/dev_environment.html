Aufsetzen einer Entwicklungsumgebung

Schritte für: Aufsetzen einer Entwicklungsumgebung auf einem Ubunut PC mit dem auch die ausführbaren Dateien für einen Windows-PC erstellt werden.

Schritt 1: Installieren der Toolchain

sudo apt-get update && sudo apt-get upgrade
sudo apt-get install mingw-w64 wine codeblocks codeblocks-contrib

Schritt 2: wxWidgets Quelltextdateien herrunterladen und für die Zielhardware kompilieren

sudo su -
cd /opt
wget https://github.com/wxWidgets/wxWidgets/releases/download/v3.0.3/wxWidgets-3.0.3.tar.bz2
tar -xvzf wxWidgets-3.0.3.tar.gz2
cd wxWidgets-3.0.3/
./configure --prefix=/usr/local/i686-w64-mingw32 --host=i686-w64-mingw32 --build=i686-linux --disable-shared --enable-unicode
make
make install

Schritt 3: Das Programm Codeblocks öffnen und die foglenden Einstellungen vornehmen

Settings > Compiler >

Selected Compiler > GNU GCC Compiler > Copy > New Compiler Name : MinGW Compiler

Selected Compiler > MinGW-w64 Compiler

Search Directories > Compiler > Add : /usr/i686-w64-mingw32/include/
Search Directories > Linker > Add : /usr/i686-w64-mingw32/lib/
Search Directories > Resource Compiler > Add : /usr/i686-w64-mingw32/include/

Toolchain Executables >
Program Files >
Compilers Installation Directory : /usr/i586-mingw32msvc/
C Compiler : i686-w64-mingw32-gcc
C++ Compiler : i686-w64-mingw32-g++
Dynamic Linker : i686-w64-mingw32-g++
Static Linker : i686-w64-mingw32-ld
Additional Paths : /usr/bin

Compiler Settings > Other Options : `/usr/local/i686-w64-mingw32/bin/wx-config --cxxflags`
Linker Settings > Other Options : `/usr/local/i686-w64-mingw32/bin/wx-config --libs`
Compiler Settings > Other Options : -static -lpthread
Compiler Settings > Policy : Use target options only
Linker Settinggs > Policy : Use target options only
Search Directory > Policy : Use target options only


Now create a new project in codeblocks, the key fields are as follows:

New Project > wXWidgets Project
wXWidgets > 3.0.x
Preferred GUI Builder > wxFormBuilder
Compiler > MinGW-w64 Compiler


Now run with wine:

wine ./youappname.exe
